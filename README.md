# Pizza-Sales-Trend
# 🍕 Pizza Sales Analysis Dashboard (Power BI)

📌 Project Overview

This project focuses on analyzing pizza sales data to uncover business insights, identify customer ordering patterns, and highlight opportunities for revenue growth through upselling and product combinations.

The dashboard was built using Power BI, with data preparation performed using SQL and Excel.

 🛠️ Data Preparation Workflow

* Extracted raw data from multiple sheets
* Used **SQL** to join **4 different raw tables** into a single master dataset
* Selected only relevant columns required for analysis
* Performed data cleaning and preprocessing in **Excel**:

  * Handled missing values
  * Standardized formats
  * Ensured data consistency
* Imported the cleaned dataset into **Power BI** for modeling and visualization

 📊 Dashboard Features

* **KPI Metrics**

  * Total Revenue
  * Total Orders
  * Average Order Value

* **Sales Trends**

  * Hourly order trends
  * Identification of peak order times (lunch/dinner)

* **Product Analysis**

  * Top-performing pizzas
  * Lowest-selling pizzas

* **Combo Analysis**

  * Identified frequently ordered pizza combinations
  * Ranking combinations using BI Ranking Filter

* **Category Insights**

  * Performance comparison across pizza categories (Vegetarian, Chicken, etc.)


🧠 Key Insights

* Peak order volume occurs during **lunch hours**, indicating high demand during this period
* Certain pizza combinations (e.g., **Vegetable + Chicken**) show strong ordering patterns
* A small number of pizzas contribute significantly to total sales
* Several low-performing pizzas present **upselling opportunities when bundled with high-demand items**


💡 Business Recommendations

* Introduce **combo offers during peak lunch hours** to increase average order value
* Bundle **low-selling pizzas with top-performing pizzas** to improve their sales
* Leverage popular combinations (e.g., Vegetable + Chicken) for targeted promotions
* Optimize menu positioning based on demand trends
